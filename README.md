C snake game realized on list

## What is list?

List is a data structure where each node is connected to 'next' one, from the first to the last.

![visualization](list.png)
 
## How to use it

Compile:

```bash
	make clean && make && ./terminal-snake
```
Run:

```bash
	./terminal-snake
```
